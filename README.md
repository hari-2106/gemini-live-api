# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/9276f6c1-b847-4a68-a2fa-d921087d4fb4

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set `NEXT_PUBLIC_GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key (required for local run; the app reads this in the browser)
3. Run the app:
   `npm run dev`
